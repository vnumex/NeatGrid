# NeatGrid

NeatGrid is an Android game launcher and library manager. It provides a single interface to organize and launch both native Android games and ROMs via external emulators. 

The application is built with Kotlin and Jetpack Compose, following Material 3 design guidelines.

## Features

* Unified grid interface for local game management
* Support for launching ROMs through installed emulators (to be implemented)
* Material 3 UI implementation
